## 💡 Consejos para el Estudio

1. **Enfócate en la energía:** Muchos problemas que requieren ecuaciones diferenciales complejas en dinámica se resuelven fácil con conservación de energía
2. **Identifica el sistema:** Siempre define qué objetos están incluidos en tu sistema
3. **Revisa la fricción:** Si hay fricción, $E_m$ NO se conserva
4. **Practica con el TP2:** Es la mejor forma de afianzar conceptos

## 📝 Notas Rápidas

### Teorema de Trabajo-Energía
$$W_{neta} = \Delta K = K_f - K_i$$

### Conservación de Energía Mecánica (sin fricción)
$$E_m = K + U = \text{constante}$$

### Con Fuerzas No Conservativas
$$K_i + U_i + W_{nc} = K_f + U_f$$
